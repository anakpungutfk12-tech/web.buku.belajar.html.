📘 Buku Belajar HTML

Bab 1: Apa itu HTML?

HTML adalah bahasa dasar untuk membuat website. HTML digunakan untuk menyusun struktur halaman seperti teks, gambar, dan link.

Bayangkan HTML seperti kerangka rumah. Tanpa kerangka, rumah tidak bisa berdiri.

<h1>Halo Dunia</h1>

Kode di atas akan menampilkan tulisan besar "Halo Dunia".

Latihan:
Coba ketik ulang kode di atas di VS Code lalu buka di browser.

Bab 2: Struktur HTML

Setiap halaman HTML harus memiliki struktur dasar.

<!DOCTYPE html>

<html>

<head>

<title>Judul</title>

</head>

<body>

Isi Website

</body>

</html>

Penjelasan:

  • html → pembungkus utama
  • head → informasi
  • body → isi website
Latihan:
Buat file HTML baru dan tulis struktur ini.

Bab 3: Tag dan Paragraf

Tag digunakan untuk menampilkan teks.

<p>Ini paragraf</p>

Tag memiliki pembuka dan penutup.

Latihan:
Buat 3 paragraf berbeda di halaman HTML.

Bab 4: Link dan Gambar

<a href="https://google.com">Google</a>

<img src="gambar.jpg">

Link digunakan untuk berpindah halaman. Gambar digunakan untuk menampilkan visual.

Latihan:
Tambahkan 1 gambar dan 1 link.

Bab 5: Project Akhir

<!DOCTYPE html>

<html>

<head>

<title>Website Saya</title>

</head>

<body>
